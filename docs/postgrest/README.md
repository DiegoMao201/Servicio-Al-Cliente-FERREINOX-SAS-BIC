# PostgREST

Esta carpeta concentra los archivos operativos y de referencia de PostgREST para no dejarlos regados en la raiz.

- `sync_official_postgrest.py` vive en `tools/postgrest/`.
- Los `.txt` y `.sql` de soporte quedaron en esta carpeta.
- Ejecuta el sincronizador desde la raiz del repo con `python tools/postgrest/sync_official_postgrest.py`.
